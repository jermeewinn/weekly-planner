# Weekly Planner

The purpose of this challenge was to build out an app's stylings and back-end functions using 3rd-party APIs. The planner is color-coded so that you know what is in the past, the present, and the future. 

## Built with:
- HTML
- CSS
- CSS Bulma
- JavaScript
- jQuery
- moment.js
